indexterm (db.indexterm.endofrange)

indexterm — Identifies the end of a range associated with an indexed term.

Synopsis

  • empty

Attributes

Common attributes and common linking attributes.

Additional attributes:

  • class (enumeration)
    • “endofrange”
  • startref (IDREF)

Required attributes are shown in bold.

Description

An “end of range” indexterm marks the end of a range. It must have an associated “start of range” indexterm. The resultant index entry applies to the entire range. See indexterm (db.indexterm.singular).

Processing expectations

See indexterm (db.indexterm.singular).

Attributes

Common attributes and common linking attributes.

class

Identifies the class of index term

Enumerated values:
“endofrange”

The end of a range
startref

Points to the start of the range
