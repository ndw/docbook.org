ISO Latin 1 Character Entities (%isolat1;)

%isolat1; — ISO Latin 1 Character Entities

Synopsis

The %isolat1; parameter entity includes the ISO character entities with the public identifier: 

  ISO 8879:1986//ENTITIES Added Latin 1//EN

Description

ISO and the W3C are cooperating to provide an official set of character entity references for XML. A description of this set is available at http://www.w3.org/2003/entities/iso8879doc/isolat1.html
