Common attributes

Description

Fax is a fax number in an address.

Processing expectations

Formatted inline. Sometimes suppressed. In an Address, this element may inherit the verbatim qualities of an address.

Parents

These elements contain fax: address.

Children

The following elements occur in fax: emphasis, indexterm, inlinegraphic, inlinemediaobject, link, olink, remark, replaceable, subscript, superscript, trademark, ulink.
