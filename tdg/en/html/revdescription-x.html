revdescription — A extended description of a revision to a document

Synopsis

Content Model

revdescription ::=
((%revdescription.mix;)+)

Attributes

Common attributes

Description

The RevDescription associated with a revision is a summary of the changes made in that revision. RevDescription is intended for long, complete summaries. For a simple text-only summary, see RevRemark.

Processing expectations

May be formatted inline or as a displayed block, depending on context. Sometimes suppressed.

Parents

These elements contain revdescription: revision.
