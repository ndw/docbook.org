Description

A Caption is an extended description of a MediaObject. Unlike a TextObject, which is an alternative to the other elements in the MediaObject, the Caption augments the object.

Processing expectations

Formatted as a displayed block.

Parents

These elements contain caption: mediaobject, table.

Children

The following elements occur in caption: bibliolist, blockquote, calloutlist, caution, formalpara, glosslist, important, itemizedlist, literallayout, note, orderedlist, para, programlisting, programlistingco, screen, screenco, screenshot, segmentedlist, simpara, simplelist, tip, variablelist, warning.
